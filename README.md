# Developer Portfolio AI

Small fullstack landing page that presents me as a developer and includes a working contact form, backend API, email sending and AI helper section.

## Stack

### Frontend
- React
- TypeScript
- Vite
- SCSS Modules
- Axios

### Backend
- Node.js
- Express
- Nodemailer
- Yandex SMTP

## Features

- Responsive developer landing page
- About section
- Experience/projects section
- AI workflow section
- Contact form
- Form validation
- Loading / success / error states
- Backend API for form submission
- Email notification to website owner
- Email copy to user
- AI Summary Helper endpoint
